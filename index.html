精簡方向規則版｜核心流程

大局背景 → 主判／次判市場關係 → P位置 → Q Trigger → 大局障礙 → 最終注碼

取消獨立Direction Permission Gate；方向規則直接包含喺主判 × 次判市場關係。

四種主判 × 次判市場關係

  1. 雙同向：只做共同方向。雙健康最高1注；同向有弱勢最高0.5注。反共同方向禁止，即使去到W／D／4H P1都唔直接反向。
  2. 方向衝突：順主判優先，最高通常0.5；順次判逆主判只當局部反彈／回調，以高質P1＋Q3為主，最高通常0.25。
  3. 包含轉換：可順已確認嗰層方向，P1／P2＋Q3最高0.5；逆已確認方向只等Transition層真正P1＋Q3反向試單，最高0.25。
  4. 雙轉換／橫行：只做邊界，中間位不做；P1主要邊界＋Q3最高0.5，普通P1邊界0.25，清晰P2邊界＋Q3為0.25。

最簡單方向規則

雙同向：只順向，禁止反向。

方向衝突：順主判優先；逆主判要求更高位置並降級。

包含轉換：順已確認方向可以做；去到Transition層P1先可反向試單。

雙轉換：只做邊界，唔做中間。

P位置級別

  • P1：W／D重大支持阻力、大局背景／主判真正主結、主判大型區間邊界、會改變主判Market State嘅重大結構位、多個HTF結構高度重疊。
  • P2：次判主結、主判次結、工作結構Break後首次Retest、重要Swap／Impulse Origin、實際結構＋Mon H/L／Asia／OPR／PDH／PDL。
  • P3：次判次結、入場TF主結、普通局部結構＋Session Liquidity、Fib＋普通Swap、低TF區間邊界。
  • P4：橫行中間、純Fib、純Asia／OPR、純2B冇結構、追突破、前方立即撞重大支持阻力。

P1順風只係Context，唔會將實際P2硬升做P1。

Q Trigger

Sweep → Reclaim → Weak Retest

  • Q3：有效Sweep＋有效Reclaim＋Retest明顯弱過Reclaim＋合理交易空間。
  • Q2：核心成立，但Retest較快／稍深、Momentum普通、Volume普通、Follow-through不足等。
  • Q1：冇真正Sweep／Reclaim、Retest快＋深＋強、Reclaim被完全否定、空間不足。

Price Action優先過Volume

價格推進效率 → 深度 → K線力度 → 時間 → Volume

Volume只係輔助證據，唔可以override Price Action。

Asia／OPR 2B角色

2B係Trigger優勢，唔係市場方向優勢。

  • 可以Q2 → Q3。
  • 有結構基礎嘅P3 → P2。
  • 可以強化P2可信度。
  • 唔可以創造反向交易權。
  • 唔可以P2 → P1、救P4或推翻大局障礙。

大局障礙修正

  • 障礙仍遠：到障礙前有完整R:R，按原Matrix。
  • 接近但仍有空間：降一級：1 → 0.5 → 0.25 → 0。
  • 已處於大局障礙區內仍順原方向延伸：P1 Q3最多0.5；P2 Q3最多0.25；P3／P4為0。
  • 空間不足：直接Skip。

Limit Entry｜Entry-time Q vs Post-entry Q

  • 深，但仍弱：結構未失效，繼續持有。
  • 深＋強：Q3可降Q2；Thesis未Invalid時，返入場／BE區可考慮減半。
  • Thesis正式失效：關鍵invalidation被接受突破、原2B完全失敗、反方向新結構成立 → 即時全走，唔等BE。

第二次確認

放棄第一個Limit Entry → 等返原P1／P2區 → 再打一個右側

例如Sweep Micro High／Low → Reclaim → Weak Retest再入場。

五項真正Hard Veto

  1. P4／中間位／追價
  2. 冇有效Sweep或Reclaim
  3. Retest明顯快＋深＋強，已否定Reclaim
  4. 第一真實Target前空間不足
  5. 違反交易時間／總風險上限

最終決策流程

大局背景 → 主判 × 次判市場關係 → 可做方向／基本上限 → P → Q → 大局障礙 → 最終注碼

V3.3最核心六句

  1. 雙同向只做共同方向，W／D大位都唔直接反向。
  2. 衝突市場可以兩邊部署，但順主判優先；逆主判要求P1並降級。
  3. 包含轉換時，可順已確認方向；去到Transition層P1大位先可捕反方向，而且最高通常0.25。
  4. 雙轉換只做邊界，不做中間。
  5. P決定位置資格，Q決定執行質素；Asia／OPR 2B唔可以推翻市場關係。
  6. 大局障礙只會降級或取消，不會令交易升級。
