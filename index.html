核心流程

大局背景 → 主判斷 × 次判斷 → 交易優先方向 → 位置級別 → Trigger Model → Trigger質素 → 大局障礙修正 → 最終注碼

七種市場狀態

  • 健康升勢
  • 弱升勢
  • 轉換中－偏升
  • 轉換中－中性
  • 轉換中－偏跌
  • 弱跌勢
  • 健康跌勢

市場狀態由該時間框架自身結構決定；優先方向先由主判 × 次判互動決定。

一注唯一權限

主判＋次判健康同向 ＋ P1／P2 ＋ Q3 ＋ 冇大局障礙 ＋ 冇硬性否決。

P1／P2最重要修正

  • P1包括大局背景主結、主判主結、主判大型區間邊界，以及會改變主判／次判市場狀態嘅主結突破回測。
  • 次判主結只有同大局／主判重大位置直接空間重疊，先升做P1。
  • 價格已離開大局重大區，之後回抽新形成次判主結，維持P2，只係有P1背景順風。
  • P2邊緣＋Q3降一級；P2邊緣＋Q2偏向Skip，本App採用保守0注。
  • 位置級別睇空間重疊，不是故事重疊。

Trigger Model A｜Liquidity Reversal

Sweep → Reclaim → Weak Retest

  • 有效Sweep。
  • 有效Reclaim。
  • Retest明顯弱過Reclaim。
  • 第一真實目標前有合理R:R。

Trigger Model B｜Breakout Continuation

Breakout → Acceptance → Weak Retest

  • Breakout位置有意義。
  • 有真正Acceptance，而唔係插穿即收返。
  • Breakout有有效推動。
  • Retest弱過Breakout。
  • Retest有結構承接；唔需要精準回原Breakout線。
  • 第一真實目標前有合理R:R。

冇Sweep唔代表0注。 Breakout延續單只要Model B完整成立，就可以正常評Q3／Q2。

OPR／亞洲盤2B

  • 高質門檻：6項條件中至少5項成立。
  • 其中一項加分條件係「沒有頂底雙邊掃」。
  • 有原有結構基礎時可以P3→P2。
  • Trigger質素仍由Model A／B獨立評級。
  • 唔可以P2→P1。
  • 唔可以救P4。
  • 唔可以推翻大局障礙。
  • 唔可以突破市場注碼上限。

大局障礙

  • 障礙仍遠、有完整R:R：按原矩陣。
  • 接近障礙但仍有空間：注碼降一級。
  • 到障礙空間不足：0注。
  • 已處於障礙區內仍順原方向延伸：P1＋Q3最多0.5；P2＋Q3最多0.25；其他0。
  • 特例：主判轉換傾向同次判趨勢相反，而你正順次判做局部回調trade；一到主判潛在支持／阻力區，停止開新同方向倉，直接0注。

五項硬性否決

  1. P4中間位或追價。
  2. 所選Trigger Model核心邏輯失效；Model A／B兩套都唔可以靠缺失核心條件硬做。
  3. Retest明顯快、深、強，否定原Reclaim或Breakout推動。
  4. 第一個真實目標前冇最低可接受R:R。
  5. 違反交易時間或總風險上限。
