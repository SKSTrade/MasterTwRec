MASTER TRADE V3.5

核心架構

Market State → P位置 → Trigger Model → Q質素 → 大局障礙 → Matrix注碼

  • Market State決定方向及風險上限。
  • P只評實際Entry位置。
  • Q只評Trigger質素。
  • 大局背景只可以限制／降級，唔可以升級。
  • 位置好唔等於Trigger好；Trigger好亦唔等於位置好。

主判 × 次判市場關係

  • 健康同向：最高1注；1注只限P1／P2＋Q3＋無重大障礙。
  • 同向有弱勢：最高0.5注。
  • 包含Transition：最高0.5注；偏向唔等於已確認Trend。
  • 方向衝突順主判：最高0.5注。
  • 方向衝突逆主判：P1 Q3最多0.5；P2 Q3通常0.25；P3／P4不做。
  • 弱勢衝突：只做重要邊界，中間位不做，最高0.5注。
  • 特殊逆向｜只順大局、逆主判＋次判：只做P1＋Q3，最高0.25注；其餘全部0。

特殊逆向｜只順大局、逆主判＋次判

當交易方向順大局背景，但同時逆主判＋次判兩層現行同向趨勢，唔當普通方向衝突處理。

  • 唯一交易資格：P1大局核心位置＋Q3反轉Trigger。
  • 最高注碼：0.25注。
  • P1 Q2、P2、P3、P4、Q1全部0注。
  • Trigger優先Model A｜Sweep → Reclaim → Weak Retest。
  • 大局核心位置只決定P1；大局方向只提供背景；兩者都唔屬於Trigger Q加分。
  • OPR／Asia 2B如果已作Model A本體，唔會再重複增加Q；位置已係P1亦唔需要再做P3→P2升級。
  • 終止：主判或次判其中一層Market State有效改變，令關係不再係「逆主判＋次判兩層同向趨勢」，立即退出特殊逆向，按最新Market State重新跑正常Matrix。

口訣：只順大局、逆主判＋次判 → 只做P1＋Q3，最高0.25注。

P位置評級

  • P1：重大W／D支持阻力、大局／主判主結、主判大型Range邊界、改變主判／次判Market State嘅重要結構Break後首次Retest等。
  • P2：次判主結、主判次結、重要Range Edge、有意義Breakout後首次Retest、清晰Swap／Impulse Origin等。
  • P3：普通但有效LTF結構、次判次結、Trigger TF主結、普通Range Edge等。
  • P4：Range中間、純Fib、純Liquidity、純2B冇結構、Chase、第一真實障礙令RR不足。

P2＋P1背景：Entry已離開P1後喺低級結構出Setup，Entry仍然係P2；P1只係背景。

Asia／OPR 2B位置規則

  • 2B正式歸入P評級，唔再係獨立Decision Step。
  • P3結構位＋高質Asia／OPR頂底2B，可P3 → P2。
  • 純Asia／OPR Sweep冇結構基礎，唔可以由P4創造P2。
  • 高質2B沿用6項評分；5／6或以上具位置升級資格。
  • 同一個Asia／OPR Sweep如果已係Model A核心證據，唔可以再當Q Bonus重複計分。

Trigger Model A｜Liquidity Reversal

Sweep → Reclaim → Retest

  • Q3：有意義Sweep＋有效Reclaim＋Weak／Controlled Retest＋合理RR。
  • Q2：核心仍成立，但有1項未致命瑕疵。
  • Q1：冇真正Sweep／Reclaim、Retest否定Reclaim、Reclaim立即失敗或RR不足。

Trigger Model B｜Breakout Continuation

Breakout → Acceptance → Retest

  • Breakout必須係有意義結構；純Micro High／Low不足以構成正式Model B。
  • Q3：有意義Breakout＋真正Acceptance＋Weak／Controlled Retest＋結構支持＋合理RR。
  • Q2：核心仍成立，但Momentum／Follow-through／Acceptance／Retest其中1項有未致命瑕疵。
  • Q1：位置唔重要、False Breakout、冇Acceptance、Retest否定Breakout或RR不足。

Q2 → Q3修正

  1. 原本只係Q2，而且只有1項輕微／未致命瑕疵。
  2. 至少2項獨立加分證據。
  3. 至少1項直接補強原本嗰個瑕疵。
  4. 同一證據只計一次，禁止Double Count。

Q1永遠唔可以靠加分升級。之後出現新Reclaim／Breakout／Acceptance，只可以當新Trigger重新評級。

Hard Veto

  • P4。
  • Chase。
  • 唔符合Model A或Model B／Trigger核心失效。
  • Model B Breakout位置冇重要性或冇Acceptance。
  • Retest快／深／強到否定原推動。
  • 第一真實Target令RR不足。
  • 違反交易時間或單日／總風險限制。
