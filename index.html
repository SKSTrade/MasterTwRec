MASTER TRADE DECISION MATRIX V3.2

核心流程

大局背景 → 主判／次判市場狀態 → Direction Permission → P位置 → Q Trigger → 大局障礙 → 最終注碼

方向權限決定有冇資格做呢個方向；P × Q只決定有權做之後，值唔值得做同打幾多。

Direction Permission｜方向權限 Gate

  • 正常方向權限：例如主判健康升＋次判健康升做Long；最高1注。
  • 限制方向權限：同向有弱勢／Transition，或方向衝突但順主判；最高通常0.5注。
  • 無方向權限：例如主判＋次判健康升，普通位置想Short；正常P1／P2＋Q3都係0。

P/Q唔可以創造方向權限。

大局P1反轉例外

原本方向權限＝0，但同時符合以下兩項，可開0.25注試單：

  1. 真正大局P1：W／D重大支持阻力、大局／主判真正主結、大局Range真正邊界等。
  2. Q3級右側反轉確認：Sweep → Reclaim → Weak Retest完整成立。

無權限＋真正大局P1＋Q3右側反轉 → 0.25注

P2＋Q3、純Asia／OPR、純Fib、冇右側確認都唔可以用例外。

P位置

  • P1：W／D重大S/R、大局背景／主判真正主結、大型Range邊界、會改變主判Market State嘅主結Break後首次Retest、多個HTF結構高度重疊。
  • P2：次判主結、主判次結、工作結構Break後首次Retest、重要Swap／Impulse Origin、實際結構＋Mon H/L／Asia／OPR／PDH／PDL。
  • P3：次判次結、入場TF主結、普通局部結構＋Session Liquidity、普通Fib＋Swap。
  • P4：Range中間、Chase、純Fib、純Asia／OPR、純2B冇結構、前方即撞重大障礙。

P1順風只係Context，唔會將實際P2硬升做P1。

Q Trigger

Sweep → Reclaim → Weak Retest

  • Q3：有效Sweep＋有效Reclaim＋Retest明顯弱過Reclaim＋合理交易空間。
  • Q2：核心仍成立，但Retest稍快／稍深、Reclaim普通、Volume普通、Follow-through不足等。
  • Q1：冇有效Sweep／Reclaim、Retest快深強、Reclaim完全否定、反向Micro Structure明顯建立或R:R不足。

Price Action優先過Volume

價格推進效率 → 深度 → K線力度 → 時間 → Volume

Volume只係加分證據，唔可以override Price Action。Retest Volume細，但價格跌得快、深、連續大實體，仍然係強Retest。

大局障礙修正

  • 障礙仍遠：完整R:R，按原Matrix。
  • 接近但仍有空間：降一級，1 → 0.5 → 0.25 → 0。
  • 已處於障礙區內仍順原方向延伸：P1 Q3最多0.5；P2 Q3最多0.25；P3／P4為0。
  • 空間不足：0注。

Asia／OPR 2B

2B係執行優勢，唔係方向優勢。

  • 可以Q2 → Q3。
  • 有結構基礎嘅P3 → P2。
  • 可以強化P2可信度。
  • 唔可以創造Direction Permission。
  • 唔可以P2 → P1、0注 → 0.5、救P4或推翻大局障礙。

Limit Entry｜Entry-time Q vs Post-entry Q

  • 深，但仍弱：結構未失效，可以持有。
  • 深＋強：Q3可降Q2；Thesis未Invalid時，返入場／BE區可考慮減半。
  • Thesis正式Invalid：關鍵失效位被接受突破、原2B完全失敗、反方向新結構＋Follow-through，立即全走，唔等BE。

第二次確認

第一個Limit Entry因Retest變強時：

放棄第一次掛盤 → 等返原P1／P2區 → 再等低TF右側確認

即係再打一個Sweep Micro High／Low → Reclaim → Weak Retest嘅右側Trigger。

最終公式

Market State → Direction Permission → P → Q → 大局障礙 → 最終注碼

Direction Permission係Gate；最終注碼永遠取所有限制中最低值。

V3.2最核心五句

  1. 方向權限係Gate，P/Q唔可以創造方向權限。
  2. 正常無權限＝0注。
  3. 唯一例外：真正大局P1＋Q3右側反轉，可以開0.25注試單。
  4. 大局P1順風只係背景，唔會將P2自動升P1。
  5. 正面撞大局障礙時，就算順方向P2＋Q3，都可能降到0.25甚至0。
