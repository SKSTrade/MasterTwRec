Market State硬規則

主結未實收穿，一律仍屬原趨勢，最多降成弱勢。

  • Wick穿主結 → 唔算Transition。
  • Sweep主結後收返 → 唔算Transition。
  • 次結破／深回調 → 最多弱勢。
  • 只有有效實收穿主結，先由Trend進入Transition，再判偏升／中性／偏跌。

主判 × 次判方向規則

  • 雙同向：只做共同方向。雙健康最高1；同向有弱勢最高0.5；反向固定0。
  • 方向衝突：順主判優先，P1／高質P2＋Q3最高0.5；順次判逆主判P1＋Q3正常最高0.25。
  • 逆主判P2：正常0；只要主判仍係趨勢、次判已不再同主判同向，而且P1順風仍有效，P2＋Q3可最高0.25。唔需要主判次結先被突破。
  • 包含轉換：順已確認方向P1／P2＋Q3最高0.5；反向正常以Transition層真正P1＋Q3為主。若主判仍係趨勢、次判已不再同主判同向＋有效P1順風，P2＋Q3亦可0.25。
  • 雙轉換／橫行：只做邊界，中間0。

Setup Type

Type A｜高質OPR／Asia 2B

方向一致＋現有6項評分5／6或以上。原始P3可獲P2-effective待遇；P2唔升P1；P4救唔到。

Q2只有唯一問題係Sweep／Reclaim質素邊緣，而且2B足以確認該部分，先可Q2 → Q3。Retest強、空間不足、Setup被否定都救唔到。

Type B｜普通Sweep & Reclaim

普通Swing、PDH／PDL、結構高低位Sweep等。冇任何P／Q特殊升級。

Type C｜No Sweep

冇Liquidity Sweep，只考慮P1＋高質右側確認：P1 Rejection＋Micro Structure Break＋Weak Retest＋合理空間。P2／P3普通No-Sweep Setup＝0。

P位置｜大局實際重疊 vs P1順風

大局實際重疊＝我依家喺邊入場。 Entry zone本身同HTF真實價格結構有空間交集，可以直接影響P級。

P1順風＝呢段Move由邊個大位引發。 只影響第一段反轉＋第一次回調Setup嘅方向權重，永遠唔會將P2升P1。

裸K測試：Hide晒Fib、OPR、Asia、Mon H/L、PDH／PDL後，仍會獨立畫出該區，先算真實結構。

P1順風有效期

只計P1直接引發嘅第一段反轉＋第一次回調Setup。

完成有效回調、正式確認新趨勢、第一個LH／HL循環完成、轉橫行、或者關鍵結構被反向Reclaim後，P1順風失效。

Q質素

取消Trigger加分證據逐項計分，避免Double Counting。

  • Type A／B Q3：Reclaim有效、Retest明顯弱、Setup冇即時被否定、第一真實目標有合理空間。
  • Q2：核心仍成立，但Retest稍快／深／強、Reclaim一般或結構確認未完整。
  • Q1：Reclaim失敗、Retest快＋深＋強到否定Setup、反方向Micro Structure主動建立、或者空間不足。
  • Type C：改睇P1 Rejection、右側Micro Break、Weak Retest同空間。

價格推進效率 → 深度 → K線力度 → 時間 → Volume

次判轉換中性｜Range 25%修正

  • Long喺底25% → 正常Matrix；唔喺底25% → 降一級。
  • Short喺頂25% → 正常Matrix；唔喺頂25% → 降一級。
  • 真正Range正中／P4 → 0。

大局障礙｜Entry vs Target

  • Entry Obstacle：第一真實障礙太近。RR不足＝0；仍可交易但空間明顯壓縮＝降一級。
  • Target Obstacle：到障礙前已有完整最低RR，例如2R；唔影響Entry注碼，只影響TP、Runner同突破預期。
  • 已處障礙區內仍順向延伸：P1 Q3最多0.5；P2 Q3最多0.25；P3／P4為0。

Limit Entry／第二次確認

  • Retest變深但仍弱 → Hold。
  • Retest變深＋強，Q3降Q2但未Invalid → 回Entry／BE附近考慮減半。
  • Thesis Invalid → 即時走，唔等BE。
  • 第一個Retest變強 → 放棄第一次Limit，等原P1／P2再做第二次右側確認；新Trigger可重新評Q3。

五項Hard Veto

  1. P4／Range正中／追價
  2. Setup核心確認失敗
  3. Retest快＋深＋強到否定Setup
  4. 第一真實目標前RR不足
  5. 違反交易時間／總風險上限

V3.4最核心十句

  1. 雙同向只做共同方向，禁止反向。
  2. 衝突／轉換市場先容許反向劇本，但反向位置要求更高、注碼更低。
  3. 主結未實收穿，一律最多只係弱勢；實收穿先叫轉換。
  4. Type A高質OPR／Asia 2B可令P3獲P2-effective待遇。
  5. 普通Sweep-Reclaim無任何特殊升級。
  6. No Sweep只考慮P1＋高質右側確認。
  7. P1順風唔升P，只負責第一段反轉＋第一次回調Setup嘅方向權重。
  8. 次判轉換中性時，Long優先底25%、Short優先頂25%；唔喺相應區域降一級。
  9. 大局實際重疊＝你依家喺邊入；P1順風＝呢段Move由邊度引發。
  10. P決定位置、Setup Type決定模式Edge、Q決定執行質素；三者唔重複計分。
